# Skills for Claude

Custom instructions that extend Claude's capabilities. Each skill is self-contained with copy-paste ready instructions.

## Available Skills

| Skill | Description |
|-------|-------------|
| [no-ai-writing](./no-ai-writing/) | Makes Claude write like a human by eliminating AI writing patterns |

## How to Use a Skill

1. Click on a skill folder above
2. Open the skill's README
3. Follow the instructions there (usually: copy text, paste into Claude settings)

## Adding Skills to This Repo

Each skill lives in its own folder with:
- `README.md` - Copy-paste instructions for all users
- `SKILL.md` - Full version with external references (for Claude Code)
- `CLAUDE.md` - Drop-in file for Claude Code projects
